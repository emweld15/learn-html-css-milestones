Relationship Between Tags:

The hyperlink tag is a child of the paragraph tag.

Attributes and Values of the Hyperlink Tag:

  • href: https://vestique.com/
  • id: clothing-link
  • class: clothes
  • target: _blank

Purpose of the Attributes:

The "href" attribute gives the URL of the page the link to go to.

The "id" attribute gives a unique id for an element.

The "class" attribute refers to a class in the stylesheet.

The "target" attribute indicates where the linked element will open. Here, it will open in a new tab.
